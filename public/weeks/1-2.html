ASSIGNMENT: Google Site / Sitemap

About

Rather than going with google sites, I decide to make a simple html / css page and host it on neocities, a modern reimagining of geocities. I wanted to capture an old internet feel, thus the intent is for the site to look a bit hobbled together. The link to the code can be found in the navbar. To create the sitemap itself, I used draw.io. I chose draw.io because it is free and I have a lot of experience using it. The elements themselves are shapes used in UML diagrams. I chose this structure as I found it reminiscent of some of my earlier computer science courses, where the professors would have a very simple and directly outline for the course. On the homepage there is a button / link to a given week of the course material. Any assignments related to that will be included on that page. Each assignment or project will have its own section on the weekly page.
